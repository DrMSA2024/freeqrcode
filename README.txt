QR Code Generator

Overview:
This project is a QR Code Generator that allows users to select different types of QR codes (URL, PDF, WhatsApp, Instagram, Email, etc.) and generate corresponding QR codes easily.

Features:
- Supports multiple QR types.
- Dynamic input fields based on selection.
- Generates QR codes using qrcode.js.
- User-friendly interface.

Installation:
1. Clone the repository:
   git clone https://github.com/your-username/qr-code-generator.git
2. Navigate to the project folder:
   cd qr-code-generator
3. Open index.html in a browser.

Usage:
1. Select the QR type.
2. Enter the required information.
3. Click 'Generate' to create the QR code.
